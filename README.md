# putfood.in
## Mealtime Collaboration

Organise your mealtime by emailing household members to get their dinning preference.

## Requirements

Install vagrant, virtualbox and ansible on whatever OS floats your boat. No judgement.

## Usage

```
vagrant up
vagrant ssh
cd /opt/putfood.in
npm start
```
