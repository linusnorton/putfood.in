# putfood.in
## Mealtime Collaboration

Organise your mealtime by emailing household members to get their dinning preference.

## Requirements

Vagrant, virtualbox and ansible.

## Usage

```
vagrant up
vagrant ssh
cd /opt/putfood.in
npm start
```
